Binäre Türme von Hanoi

Binäre Zahl: 0000
Anzahl der Schritte: 0
Anzahl der Scheiben: 4

Die binäre Zahl ist so lang wie es Scheiben gibt.

Bei jedem Schritt wird die Zahl um 1 erhöht nach der binären Logik.

Jede Scheibe ist an eine binäre Stelle gekoppelt.

Die größte Scheibe stellt die erste Stelle dar, usw.

Springt beim nächsten Schritt eine Zahl von 0 auf 1, wird diese Scheibe verschoben.

1.) Es gelten die normalen Türme von Hanoi Regeln, dass jede Scheibe nur an eine Stelle kann, wenn sie nicht größer ist, als die Scheiben, die schon drauf liegen.

Es wird immer ein Stapel gesucht, wo es passen kann.

2.) Es gibt nur eine Extra Regel für die kleinste Scheibe, dass sie immer nach rechts wandert, wenn ihre Zahl auf 1 geht. Kann sie nicht mehr nach rechts wandern, fängt sie wieder links vorne an.
